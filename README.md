# Battery-Data-Engineering-NSR2022

# Battery Data Engineering: Feature Extraction from LDMs
This repository contains the data processing pipeline used in our study published in **National Science Review (2022)**.

## 📌 Overview
In the original study, we developed Local Density Models (LDMs) to understand battery interphase evolution. This toolkit automates the transition from raw DFT outputs (e.g., VASP OUTCAR) to structured feature matrices for physical analysis.

## 🛠 Key Features
- **Data Curation:** Automated parsing of large-scale computational datasets.
- **Feature Engineering:** Extraction of local environment descriptors (coordination numbers, bond lengths).
- **Physics-driven Filtering:** Noise reduction algorithms based on thermodynamic constraints.

## 🚀 Skills Demonstrated
- **Python Libraries:** Pandas, NumPy, Matplotlib, SciPy.
- **Computational Logic:** Transforming high-dimensional physical states into computable descriptors.
- **Reproducibility:** A standardized pipeline to ensure data integrity for AI4S research.
